# Wazen CFO Intelligence Agent V13.6

## هدف التحديث
تحويل صفحات القرار والتحليل من عرض أرقام متفرقة إلى تجربة أكثر تنفيذية:
- لا يظهر حكم على بطاقة القرار دون معيار أو مصدر حكم.
- لا تتكرر خريطة القطاع بين إعداد النشاط وجاهزية التحليل.
- لا تعرض معادلة تكلفة البضاعة بشكل بدائي داخل الواجهة؛ تستخدم داخلياً وتظهر كمنهجية تحقق عند الحاجة.
- التحليل الأفقي يقرأ ميزان مراجعة سنة مقارنة عند رفع ميزانين حالي وسابق.
- الجداول التفصيلية أصبحت خلف بطاقات تحليلية وعبارات فتح أكثر مهنية.

## أهم التعديلات

### 1. مؤشرات القرار
- أضيف لكل بطاقة:
  - الدليل المستخدم.
  - معيار الحكم.
  - مصدر المعيار.
  - الإجراء المقترح.
- حذف قسم "ماذا أراجع بعد القرار؟" واستبداله بجدول منهجية يظهر فقط عند الحاجة.
- إضافة Expander باسم "منطق احتساب بطاقات القرار" لتفسير لماذا ظهر الحكم.

### 2. التحليل الرأسي والأفقي
- إضافة مقارنة سنة بسنة عند رفع ميزان مراجعة حالي وسابق.
- مقارنة قائمة الدخل بين السنتين: الإيرادات، تكلفة الإيراد، مجمل الربح، مصاريف التشغيل، EBITDA، صافي الربح.
- مقارنة أكبر تغيرات الحسابات في المركز المالي.
- إضافة بطاقات ملخص قبل الجداول.
- استبدال عبارة "افتح التفاصيل حسب الحاجة" بعبارة "التحليلات التفصيلية".

### 3. تكلفة البضاعة / تكلفة الإيراد
- إخفاء المعادلة البدائية من الواجهة الرئيسية.
- استخدام المعادلة كمنهجية تحقق داخل Expander فقط.
- عرض حالة تكلفة الإيراد كرسالة مهنية: مكتملة / تحتاج تحقق / تحتاج تحديد.

### 4. جاهزية التحليل
- إزالة تكرار خريطة تفكير القطاع من صفحة جاهزية التحليل.
- استبدال جدول الملفات المكرر بجدول "مصادر النموذج المعتمدة".

### 5. CSS وتجربة العرض
- إضافة بطاقات تحليل V13.6.
- إضافة عرض معيار الحكم داخل بطاقات القرار.
