# Wazen CFO Agent V12.2 — Core Fixes

## لماذا هذه النسخة؟
هذه النسخة تعالج ملاحظات الاختبار على V12.1 قبل البدء في المرحلة الثانية.

## أهم الإصلاحات

### 1. التنقل بعد الإجراءات
- عند حفظ سياق النشاط ينتقل التطبيق مباشرة إلى رفع الملفات.
- عند بناء النموذج المالي ينتقل مباشرة إلى جاهزية التحليل.
- تم تحويل التنقل إلى Session State بدل بقاء المستخدم في نفس الصفحة بعد رسالة النجاح.

### 2. تصنيف المصاريف
- تفعيل التصنيف الذكي عبر OpenAI عند توفر المفتاح، مع fallback لقواعد مالية محسّنة.
- توسيع قاموس التصنيف العربي لتقليل Other Opex.
- إضافة فلاتر واضحة أعلى لوحة التصنيف: بحث، تصنيف مقترح، تصنيف معتمد، نوع تكلفة، منخفضة الثقة، Other Opex، البنود الكبيرة.
- التركيز على مراجعة الاستثناءات بدل التصنيف اليدوي الكامل.

### 3. جاهزية التحليل
- استبدال إضافة الملفات العامة ببطاقات توصية لكل ملف ناقص.
- يمكن رفع الملف المقترح من نفس بطاقة التوصية، مع ضمه مباشرة للقراءة الحالية.
- عند إضافة ملف جديد يتم تحديث السيولة وإعادة بناء النموذج عندما يكون ممكنًا.

### 4. التشخيص التنفيذي
- إعادة كتابة رسالة CFO بحيث لا تظهر تناقضات مثل ربح مع مصاريف أعلى دون تفسير.
- فصل قراءة الربحية عن قراءة النقد والتحصيل.
- إضافة تشخيص يشرح: هل الشركة رابحة؟ هل الربح كافٍ؟ هل السيولة مكتملة؟ وما الإجراء العملي القادم.

### 5. السيناريوهات
- ضبط السلايدرز بحيث يكون رقم 0 في المنتصف.
- تحويل المدخلات إلى تغيرات عن الوضع الحالي وليس نسبًا جامدة فقط.

### 6. مساحات التحليل
- تغيير عبارة استقرار الإيراد إلى انتظام الإيراد مع شرح معناها.
- إضافة جدول تفسير مؤشرات الإيراد بدل بطاقة سطحية فقط.
- عند نقص السيولة أو أعمار العملاء تظهر إمكانية رفع الملف من نفس التبويب.

## ملاحظة
صفحة التنبيهات والإجراءات لا تزال Foundation، وستصبح أقوى في المرحلة الثانية بعد بناء Revenue Quality & Benchmark Engine.
