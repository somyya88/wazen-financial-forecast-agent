# Wazen CFO Agent V12.8 — Sector-aware Metric Guard + Decision UX

## الهدف
تحويل التجربة من جداول ونسب طويلة إلى تجربة قرار CFO تفاعلية: القطاع يوجه التحليل، البيانات تثبت قابلية المؤشر، وحارس النسب يمنع عرض رقم بلا مصدر أو مدخلات.

## أهم الإضافات

### 1. Sector Profile Engine
- يقرأ سياق الشركة: القطاع، النشاط، نموذج العمل، قناة البيع.
- يحدد عقلية التحليل الخاصة بكل قطاع.
- يحدد المؤشرات الأساسية والملفات التي ترفع الدقة.
- يمنع منطقًا جامدًا مثل: SaaS دائمًا بلا مخزون أو تجارة دائمًا بمخزون.

### 2. Metric Catalog CMA
- كتالوج للنسب المالية ومعادلاتها.
- يوضح لكل نسبة: المدخلات، المصدر الأساسي، المصدر البديل، وملاحظة CMA.
- يستخدم كمرجع داخل صفحة كتالوج النسب.

### 3. Metric Source Guard
- لا يعرض النسبة كرقم إلا إذا توفرت مدخلاتها.
- يميز بين: أساسي، مهم عند توفر البيانات، تقديري، غير قابل للحساب، وغير قابل للتطبيق.
- يمنع عرض DSO/DPO/CCC كصفر وهمي عند غياب البيانات.
- يضيف درجة ثقة ومصدر حساب لكل مؤشر.

### 4. Financial Health Score مع Coverage/Confidence
- المؤشرات الناقصة لا تُسجل كجيدة أو سيئة.
- يتم حساب Score فقط من المؤشرات المتاحة.
- تعرض Coverage وConfidence بجانب Health Score.

### 5. Benchmark Intelligence
- المعايير القطاعية أصبحت تعرض كإرشادية فقط.
- الأولوية للمقارنة الداخلية ثم السنة السابقة ثم المعايير الإرشادية.
- لا يدخل أي Benchmark خارجي في الحكم النهائي دون مصدر وفترة وثقة.

### 6. Decision UX
- إضافة CFO Command Center في مساحة التحليل.
- عرض القرار والمخاطر والثقة قبل الجداول.
- صفحة مؤشرات القرار تعرض تغطية المؤشرات وفجوات البيانات.
- إضافة لوحة عقلية القطاع في إعداد النشاط وجاهزية التحليل.

## قاعدة النسخة
لا تعرض رقمًا إلا إذا عرفت: معادلته، مصدره، مدخلاته، ملاءمته للقطاع، ودرجة ثقته.
