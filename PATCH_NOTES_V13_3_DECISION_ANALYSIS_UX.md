# V13.3 — Decision Analysis UX

## Scope
تم تطوير الصفحات الحالية فقط ضمن مساحة التحليل:
- مؤشرات القرار
- التحليل الرأسي والأفقي
- تدقيق الأرقام أصبح داخليًا وليس صفحة مستقلة

## Key changes
- إزالة تبويب مصادر الأرقام كصفحة مستقلة لأنه كان يشوش تجربة المستخدم.
- تحويل مؤشرات القرار إلى بطاقات قرار، مع جداول تظهر عند الضغط فقط.
- تحويل التحليل الرأسي والأفقي إلى عرض بصري يبدأ من: من كل 100 ريال مبيعات.
- إبقاء الجداول داخل أقسام قابلة للفتح، مع أعمدة أكثر مهنية مثل: المعنى لصاحب العمل، الإجراء المقترح، مصدر الرقم.
- استبدال تسمية ملاحظة CMA بقراءة مهنية في الواجهة.
- توضيح قاعدة تكلفة البضاعة المباعة في الجرد الدوري: مخزون أول المدة + صافي المشتريات - مخزون آخر المدة.
