# Wazen CFO Intelligence Agent V12.0

## التشغيل
```bash
pip install -r requirements.txt
streamlit run app.py
```

## فكرة الإصدار
V12 يبدأ من تجربة مستخدم موجهة بدل صفحة واحدة. الهدف أن يتحرك المستخدم من سياق النشاط ورفع البيانات إلى تشخيص CFO وسيناريوهات وتوصيات عمل.

## الملفات المضافة
- `data_readiness_v12.py`
- `liquidity_engine_v12.py`
- `scenario_engine_v12.py`
- `action_engine_v12.py`
- `period_normalization_v12.py`
- `PATCH_NOTES_V12_FOUNDATION.md`

## أهم التحسينات
- بداية الإيجنت بسياق النشاط والقطاع والبلد.
- مركز رفع ملفات منظم.
- درجة جاهزية التحليل.
- قراءة تقرير السيولة النقدية.
- قراءة أعمار العملاء والموردين.
- جدول تفاعلي لأولويات التحصيل.
- سيناريوهات مبدئية وتفاعلية.
- مركز إجراءات وتنبيهات.
