# Wazen CFO Intelligence Agent V14.2

نسخة V14.2 تعيد بناء التشخيص التنفيذي حول أسئلة صاحب العمل الأساسية بدلاً من تقرير نصي طويل، وتصلح عرض المقارنة الأفقية ومؤشرات الاتجاه.

## أهم جديد V14.2

- تشخيص تنفيذي في 7 بطاقات قرار:
  1. أين تربح الشركة؟
  2. أين تخسر أو يتآكل الهامش؟
  3. لماذا يحدث ذلك؟
  4. ما أثره على السيولة؟
  5. هل الوضع يتحسن أم يسوء؟
  6. ما القرار المطلوب الآن؟
  7. ما الإجراء الأول؟
- شريط سلسلة الربحية بالترتيب: الإيراد، تكلفة الإيراد، مجمل الربح، مصاريف التشغيل، الإدارية، البيع والتسويق، صافي النتيجة.
- حذف جدول مصاريف الإيراد المنفصل من التحليل الرأسي لأن هذه البنود جزء من قائمة الدخل.
- إصلاح ظهور وسوم HTML داخل بطاقات المقارنة الأفقية.
- إبقاء سنة التحليل وسنة الأساس قابلة للاختيار عند رفع أكثر من سنة.

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

## تفعيل AI للقراءة التنفيذية

في Streamlit Cloud ضعي المفتاح في Secrets:

```toml
OPENAI_API_KEY = "..."
```

ثم أعيدي تشغيل التطبيق.
