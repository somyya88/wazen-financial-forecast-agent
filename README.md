# Wazen CFO Intelligence Agent V13.7

نسخة V13.7 تركز على تحسين تجربة القرار والتنقل داخل صفحات التحليل بعد ملاحظات الاستخدام على V13.6.

## أهم جديد V13.7

- إظهار معيار احتساب بطاقة القرار عند تمرير مؤشر الماوس فوق البطاقة.
- حذف جدول منطق القرار من العرض الرئيسي حتى لا يكرر البيانات.
- إعادة تصميم التحليل الرأسي كهيكل ربحية كنسبة من صافي الإيراد بدل صياغة "من كل 100 ريال".
- إعادة تنظيم التحليل الرأسي والأفقي داخل تبويبات احترافية.
- إبقاء الجداول التفصيلية للتدقيق فقط، وليس كواجهة تنفيذية رئيسية.

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

المعايير الحالية داخلية/إرشادية حتى يتم ربط مصادر benchmark خارجية موثقة حسب القطاع والدولة وحجم الشركة.
