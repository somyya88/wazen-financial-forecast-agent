# Wazen CFO Intelligence Agent V14.0.1

نسخة V14.0.1 تركز على جعل التشخيص التنفيذي مولدًا بالذكاء الصناعي بناءً على بيانات الشركة الفعلية، لا نصًا ثابتًا لجميع الحالات.

## أهم جديد V14.0.1

- إضافة محرك جديد: `ai_executive_diagnosis_engine.py`.
- بناء Payload مالي تنفيذي من الأرقام المحسوبة فقط: الربحية، المصاريف، جودة الإيراد، السيولة، رأس المال العامل، المقارنة، وفجوات البيانات.
- عند تفعيل خيار AI وتوفر مفتاح OpenAI، يتم توليد قراءة CFO مخصصة للشركة الحالية.
- إذا لم يوجد مفتاح OpenAI، تظهر قراءة داخلية بديلة مع تنبيه واضح بدل أن يبدو خيار AI بلا أثر.
- منع إرسال صفوف ميزان المراجعة الخام افتراضيًا؛ يتم إرسال ملخصات ومؤشرات وتجميعات فقط.
- إضافة زر داخل صفحة التشخيص التنفيذي لتحديث القراءة التنفيذية بعد تغيير الملفات أو الإعدادات.

## إعداد AI على Streamlit Cloud

أضيفي المفتاح في Secrets باسم:

```toml
OPENAI_API_KEY = "ضع المفتاح هنا"
```

ثم أعيدي تشغيل التطبيق من Manage app → Reboot app.

## التشغيل المحلي

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ملاحظة محاسبية

الحسابات والأرقام تبقى من المحركات الحتمية داخل وازن. الذكاء الصناعي لا يحسب النسب من جديد، بل يكتب التشخيص التنفيذي من القيم المرسلة إليه.
