# Wazen CFO Intelligence Agent V14.1

نسخة V14.1 تعالج ملاحظات تجربة المستخدم والتحليل التنفيذي: اختيار سنة التحليل وسنة الأساس، اختصار صفحة جاهزية التحليل، وترتيب الملخص التنفيذي الرقمي بطريقة تعكس مسار الإيراد حتى صافي النتيجة.

## أهم جديد V14.1

- اختيار صريح للسنة محل التحليل وسنة الأساس عند رفع سنتين أو أكثر.
- دعم أفضل لتحليل 3 أو 4 سنوات بدون الاعتماد على تخمين آخر ملف مرفوع.
- إعادة بناء صفحة قابلية التحليل لتكون مختصرة وبطاقات مصادر، لا صفحة رفع طويلة.
- ترتيب بطاقات التشخيص التنفيذي: إجمالي الإيراد، تكلفة الإيراد، مجمل الربح، مصاريف التشغيل، المصاريف الإدارية، البيع والتسويق، صافي النتيجة، نقاء الإيراد.
- تعديل التشخيص الداخلي وPrompt الذكاء الصناعي ليعطي قراءة قيمة لنموذج العمل بدل تكرار الأرقام الموجودة في البطاقات.
- قراءة السيولة ورأس المال العامل محاسبياً من ميزان المراجعة عند توفر الحسابات، مع إبقاء كشف البنك كمصدر لحركة النقد والـRunway.
- تحويل تنبيهات موثوقية التشخيص إلى ملاحظات مهنية مباشرة.

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ملاحظة AI

لتفعيل التشخيص التنفيذي المولد بالذكاء الصناعي، أضيفي المفتاح في Streamlit Secrets:

```toml
OPENAI_API_KEY = "..."
```
